==========
Python-GIS
==========

Motivace
--------
* Python je dnes asi nejvíce geo-pozitivní programovací jazyk
* Python baví
* Množstí knihoven a nástrojů (GDAL, PROJ4, Shapely, Fiona, RasterIO, MapServer,
  gsconfig, OWSLib, PyWPS, pycsw, ...)
* Podpora v deskopech (GRASS GIS, ArcGIS, QGIS, ...)
